Як додати фото

Покрокова інструкція для завантаження фото на GitHub

📁 Крок 1 — Завантажте фото на GitHub

  • Зайдіть на github.com/ohgolovan-code/burger1778
  • Натисніть Add file → Upload files
  • Перетягніть фото у вікно
  • У полі назви файлу напишіть: images/назва-страви.jpg — папка створюється автоматично!
  • Натисніть Commit changes

🔗 Крок 2 — Скопіюйте посилання

  • Після завантаження зайдіть у папку images/ у репозиторії
  • Клікніть на потрібне фото
  • Натисніть кнопку Raw — відкриється пряме посилання
  • Скопіюйте URL з адресного рядка браузера

✅ Правила для фото

  • Формат: .jpg або .webp (не PNG — великий розмір)
  • Розмір: до 500 КБ на фото
  • Пропорції: горизонтальне або квадратне фото
  • Назва файлу: тільки латиниця і дефіси, наприклад burger-classic.jpg

Фото завантажені на GitHub стануть доступні тут після оновлення сторінки.
